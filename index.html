Que es MySQL?

MySQL es un sistema de gestion de bases de datos relacional (RDBMS) de codigo abierto, muy utilizado para almacenar, organizar y acceder a datos. Esta basado en el lenguaje SQL, que es el lenguaje estandar para trabajar con bases de datos relacionales.

Para que sirve?

MySQL sirve para gestionar y manipular bases de datos, lo que significa que te permite guardar, organizar, consultar, modificar y eliminar informacion de forma estructurada. Es una herramienta fundamental en el desarrollo de software y sitios web.

Codigos basicos de MySQL

A continuacion, algunos comandos fundamentales:

➣ Crear una base de datos

CREATE DATABASE nombre_base;

➣ Seleccionar una base de datos

USE nombre_base;

➣ Crear una tabla

CREATE TABLE nombre_tabla (
    id INT AUTO_INCREMENT PRIMARY KEY,
    campo1 TIPO,
    campo2 TIPO
);

➣ Insertar datos en una tabla

INSERT INTO nombre_tabla (campo1, campo2)
VALUES ('valor1', 'valor2');

➣ Consultar todos los datos

SELECT * FROM nombre_tabla;

➣ Consultar datos con condicion

SELECT campo1 FROM nombre_tabla WHERE condicion;

➣ Actualizar datos existentes

UPDATE nombre_tabla
SET campo1 = 'nuevo_valor'
WHERE condicion;

➣ Eliminar un registro

DELETE FROM nombre_tabla WHERE condicion;

➣ Eliminar una tabla

DROP TABLE nombre_tabla;

➣ Mostrar todas las bases de datos

SHOW DATABASES;

➣ Mostrar tablas en una base de datos

SHOW TABLES;

➣ Describir la estructura de una tabla

DESCRIBE nombre_tabla;
